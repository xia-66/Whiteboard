# 在线白板

一个纯前端的本地白板页面，用于临时记录文本、代码和链接。

## 功能

- 从剪贴板粘贴（按钮）
- 复制全部内容
- 导出为 TXT 文件
- 一键清空
- 自动保存到 `localStorage`
- `Ctrl/Cmd + S` 手动保存
- 实时字数/行数统计

## 使用方式

1. 直接双击打开 `index.html`，或在终端运行：

```bash
start index.html
```

2. 在输入框中输入或粘贴内容，状态栏会实时显示字数和行数。

## 数据存储说明

- 数据仅保存在当前浏览器本地（`localStorage`）
- 不会上传到服务器
- 清空按钮会删除本地缓存

## 移动端适配

- 页面为全屏布局
- 小屏设备下按钮自动换行（两列/单列）
- 输入区高度按视口动态调整
